# Kanban Board (React)

A simple Kanban board built with React that allows users to manage tasks across different stages such as **To Do**, **In Progress**, and **Done**. The project demonstrates core React concepts like state management, props, component composition, and drag-and-drop interactions.

---

## Features

- Add tasks to any column
- Delete tasks
- Drag and drop tasks between columns
- Clean and responsive UI
- Simple component-based architecture

---

## Technologies Used

- React
- JavaScript (ES6)
- CSS (Flexbox)
- HTML5 Drag and Drop API
